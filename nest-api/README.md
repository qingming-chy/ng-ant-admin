Execute the sql file provided by this project [ng-antd-admin-db.sql](ng-antd-admin-db.sql)

## Project setup

```bash
$ npm install
```

## Compile and run the project

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```
